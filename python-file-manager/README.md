# Python File Manager 

## Overview
Python File Manager is a beginner-friendly project that simulates basic file explorer functionality using Python. It allows users to perform common file and folder operations through a simple menu-driven interface.

The project is designed to strengthen understanding of file handling, directory management, and core Python programming concepts such as functions, loops, and exception handling. It also demonstrates how Python can interact with the operating system using built-in modules like `os` and `pathlib`.

## Features

### File Operations

* List files and folders
* Create a new file
* Read a file
* Update a file

  * Rename file
  * Overwrite file content
  * Append new content
* Delete a file

### Folder Operations

* Create a folder
* Delete an empty folder
* Rename a folder

## Technologies Used

* Python 3
* `pathlib`
* `os`
